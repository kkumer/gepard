## Synopsis

**cffs4mc** - serving CFF values of KM models for MC


## Preparation

First make a pygepard extension library.
**Prerequisites**: C and Fortran compilers. 
```sh
make pygepards
```
Then go into pype directory and compile auxilliary library
```sh
cd pype
make optModel.so
```
